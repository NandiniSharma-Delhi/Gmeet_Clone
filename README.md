#Gmeet Clone

This project is a gmeet clone where we can organize a meet with our friends .
Features :
 Videocall
 Screenshare
 Live Chat
 Collaborative Whiteboard

Technologies used:
 Socket.io
 PeerJs
 Node.js
 Ejs
 Express.js
 Passport.js

We can make the users join a room or create a new room and share your room code for others to join.
